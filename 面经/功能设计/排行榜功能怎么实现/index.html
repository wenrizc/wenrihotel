              • дёәд»Җд№ҲйҖүжӢ©SentinelиҖҢйқһHystrixжҲ–Resilience4jдҪңдёәжөҒжҺ§ж–№жЎҲпјҹ

              • еёғйҡҶиҝҮж»ӨеҷЁеңЁдҪ зҡ„жһ¶жһ„дёӯи§ЈеҶідәҶд»Җд№Ҳе…·дҪ“й—®йўҳпјҹдёәд»Җд№ҲдёҚдҪҝз”Ёе…¶д»–ж–№жі•пјҹ
              • еёғйҡҶиҝҮж»ӨеҷЁзҡ„еҸӮж•°пјҲе“ҲеёҢеҮҪж•°дёӘж•°гҖҒдҪҚеӣҫеӨ§е°ҸпјүжҳҜеҰӮдҪ•зЎ®е®ҡзҡ„пјҹиҜҜеҲӨзҺҮи®ҫзҪ®еӨҡе°‘пјҹ
              • жң¬ең°зј“еӯҳе…·дҪ“дҪҝз”ЁдәҶд»Җд№ҲжҠҖжңҜпјҹCaffeineгҖҒGuava CacheиҝҳжҳҜиҮӘз ”ж–№жЎҲпјҹдёәд»Җд№ҲйҖүжӢ©иҜҘжҠҖжңҜпјҢзӣёжҜ”е…¶д»–жҠҖжңҜеҘҪеңЁе“Әпјҹ

    жҺ’иЎҢжҰңеҠҹиғҪжҖҺд№Ҳе®һзҺ°

    жҺ’иЎҢжҰңеҠҹиғҪеҸҜд»ҘйҖҡиҝҮ Redis зҡ„ Sorted SetпјҲжңүеәҸйӣҶеҗҲпјү е®һзҺ°пјҢеҲ©з”Ёе…¶еә•еұӮи·іиЎЁж•°жҚ®з»“жһ„й«ҳж•ҲеӯҳеӮЁе’ҢжҹҘиҜўжңүеәҸж•°жҚ®гҖӮж ёеҝғжӯҘйӘӨеҢ…жӢ¬пјҡе°Ҷз”ЁжҲ· ID е’ҢеҲҶж•°еӯҳе…Ҙ Sorted SetпјҢжҢүеҲҶж•°жҺ’еәҸпјҢж”ҜжҢҒеҝ«йҖҹжҸ’е…ҘгҖҒжӣҙж–°гҖҒжҹҘиҜўжҺ’еҗҚе’ҢиҢғеӣҙж•°жҚ®пјҢйҖӮз”ЁдәҺе®һж—¶жҖ§й«ҳгҖҒе№¶еҸ‘еӨ§зҡ„еңәжҷҜгҖӮ

    е…ій”®дәӢе®һ

    1. жҠҖжңҜйҖүжӢ©пјҡ
      • Redis Sorted SetпјҡйҖӮеҗҲеҶ…еӯҳж“ҚдҪңпјҢжҹҘиҜўе’Ңжӣҙж–°ж•ҲзҺҮй«ҳпјҲO(log N)пјүгҖӮ
      • ж•°жҚ®еә“пјҲеҰӮ MySQLпјүпјҡйҖӮеҗҲжҢҒд№…еҢ–еӯҳеӮЁпјҢдҪҶжҹҘиҜўжҺ’еәҸж…ўпјҲйңҖзҙўеј•дјҳеҢ–пјүгҖӮ
      • еҶ…еӯҳзј“еӯҳ + е®ҡж—¶еҗҢжӯҘпјҡз»“еҗҲ Redis е’Ңж•°жҚ®еә“пјҢе…јйЎҫжҖ§иғҪе’ҢжҢҒд№…жҖ§гҖӮ
    2. ж ёеҝғеҠҹиғҪпјҡ
      • ж·»еҠ /жӣҙж–°еҲҶж•°пјҡз”ЁжҲ·еҫ—еҲҶеҸҳеҢ–ж—¶жӣҙж–°гҖӮ
      • жҹҘиҜўжҺ’еҗҚпјҡиҺ·еҸ–жҹҗз”ЁжҲ·зҡ„жҺ’еҗҚгҖӮ
      • иҢғеӣҙжҹҘиҜўпјҡиҺ·еҸ–еүҚ N еҗҚжҲ–жҢҮе®ҡеҲҶж•°ж®өз”ЁжҲ·гҖӮ
      • е®һж—¶жҖ§пјҡж”ҜжҢҒй«ҳе№¶еҸ‘жӣҙж–°е’ҢжҹҘиҜўгҖӮ
    3. Redis е®һзҺ°пјҡ
      • ж•°жҚ®з»“жһ„пјҡZSETпјҢй”®дёәжҺ’иЎҢжҰңеҗҚпјҢжҲҗе‘ҳдёәз”ЁжҲ· IDпјҢеҲҶж•°дёәеҫ—еҲҶгҖӮ
      • еёёз”Ёе‘Ҫд»Өпјҡ
        • ZADDпјҡж·»еҠ жҲ–жӣҙж–°еҲҶж•°гҖӮ
        • ZRANKпјҡжҹҘиҜўжҺ’еҗҚгҖӮ
        • ZRANGEпјҡиҺ·еҸ–еүҚ N еҗҚгҖӮ

    е»¶дјёдёҺйқўиҜ•и§’еәҰ

    • дёәд»Җд№Ҳз”Ё Sorted Setпјҹ
      • и·іиЎЁе®һзҺ°пјҢжҸ’е…ҘгҖҒжҹҘиҜўгҖҒиҢғеӣҙж“ҚдҪңеқҮдёә O(log N)пјҢиҝңдјҳдәҺй“ҫиЎЁпјҲO(N)пјүгҖӮ
      • еҶ…еӯҳж“ҚдҪңпјҢе»¶иҝҹдҪҺпјҢйҖӮеҗҲе®һж—¶жҺ’иЎҢжҰңгҖӮ
    • дјҳеҢ–ж–№жЎҲпјҡ
      • еҲҶзүҮеӯҳеӮЁпјҡж•°жҚ®йҮҸеӨ§ж—¶пјҢжҢүеҢәй—ҙпјҲеҰӮ 0-1000 еҲҶпјүеҲҶеӨҡдёӘ ZSETгҖӮ
      • зј“еӯҳ + жҢҒд№…еҢ–пјҡRedis еӯҳе®һж—¶ж•°жҚ®пјҢе®ҡж—¶еҗҢжӯҘеҲ° MySQLгҖӮ
      • еҺ»йҮҚеӨ„зҗҶпјҡеҲҶж•°зӣёеҗҢж—¶пјҢжҢүж—¶й—ҙжҲіжҲ– ID жҺ’еәҸпјҲз”ЁеӨҚеҗҲеҲҶж•°пјҢеҰӮ score.timestampпјүгҖӮ
    • й«ҳе№¶еҸ‘ж”ҜжҢҒпјҡ
      • Redis еҚ•зәҝзЁӢ + еӨҡзәҝзЁӢ I/OпјҲ6.0+пјүеӨ„зҗҶеӨ§йҮҸиҜ·жұӮгҖӮ
      • еҠ й”ҒпјҲеҰӮ Redis еҲҶеёғејҸй”ҒпјүйҒҝе…Қжӣҙж–°еҶІзӘҒгҖӮ
    • жӣҝд»Јж–№жЎҲпјҡ
      • ж•°жҚ®еә“пјҡз”Ёзҙўеј•еӯ—ж®өпјҲеҰӮ scoreпјүжҺ’еәҸпјҢдҪҶй«ҳе№¶еҸ‘дёӢжҖ§иғҪе·®гҖӮ
      • еҶ…еӯҳз»“жһ„пјҡеҰӮ Java зҡ„ TreeMapпјҢиҮӘе»әи·іиЎЁпјҢдҪҶйңҖжүӢеҠЁз®ЎзҗҶеҲҶеёғејҸеңәжҷҜгҖӮ
    • е®һйҷ…еә”з”Ёпјҡ
      • жёёжҲҸжҺ’иЎҢжҰңпјҡе®һж—¶жӣҙж–°зҺ©е®¶еҲҶж•°пјҢеұ•зӨәеүҚ 100 еҗҚгҖӮ
      • з”өе•Ҷжҙ»еҠЁпјҡз”ЁжҲ·з§ҜеҲҶжҺ’еҗҚпјҢеҠЁжҖҒеҲ·ж–°гҖӮ
      • и·іиЎЁпјҲSkip Listпјү жҳҜдёҖз§ҚйҡҸжңәеҢ–зҡ„ж•°жҚ®з»“жһ„пјҢеҹәдәҺжңүеәҸй“ҫиЎЁпјҢйҖҡиҝҮеӨҡеұӮзҙўеј•еҠ йҖҹжҹҘжүҫгҖҒжҸ’е…Ҙе’ҢеҲ йҷӨж“ҚдҪңгҖӮе®ғзҡ„е№іеқҮж—¶й—ҙеӨҚжқӮеәҰдёә O(log N)пјҢз»“еҗҲдәҶй“ҫиЎЁзҡ„з®ҖеҚ•жҖ§е’ҢдәҢеҸүжҗңзҙўж ‘зҡ„ж•ҲзҺҮпјҢе№ҝжіӣз”ЁдәҺйңҖиҰҒеҠЁжҖҒжҺ’еәҸзҡ„еңәжҷҜпјҢеҰӮ Redis зҡ„ Sorted SetгҖӮ
    • еҹәжң¬з»“жһ„пјҡ
      • еә•еұӮжҳҜе®Ңж•ҙзҡ„жңүеәҸй“ҫиЎЁпјҢеҢ…еҗ«жүҖжңүе…ғзҙ гҖӮ
      • дёҠеұӮжҳҜзЁҖз–Ҹзҙўеј•пјҢжҜҸеұӮиҠӮзӮ№жҳҜдёӢеұӮиҠӮзӮ№зҡ„еӯҗйӣҶпјҢйҖҡиҝҮжҢҮй’ҲиҝһжҺҘгҖӮ
      • еұӮж•°йҡҸжңәз”ҹжҲҗпјҢйҖҡеёёжңҖй«ҳеұӮеҲ°з¬¬ 0 еұӮпјҲеә•еұӮпјүгҖӮ
    • е·ҘдҪңеҺҹзҗҶпјҡ
      • жҹҘжүҫпјҡд»ҺжңҖй«ҳеұӮејҖе§ӢпјҢжІҝзҙўеј•еҗ‘еҸіжҲ–еҗ‘дёӢз§»еҠЁпјҢеҝ«йҖҹе®ҡдҪҚзӣ®ж ҮгҖӮ
      • жҸ’е…ҘпјҡйҡҸжңәеҶіе®ҡж–°иҠӮзӮ№зҡ„еұӮж•°пјҢжҸ’е…ҘеҲ°еҜ№еә”еұӮе№¶и°ғж•ҙжҢҮй’ҲгҖӮ
      • еҲ йҷӨпјҡз§»йҷӨиҠӮзӮ№е№¶жӣҙж–°еҗ„еұӮжҢҮй’ҲгҖӮ
    • ж—¶й—ҙеӨҚжқӮеәҰпјҡ
      • е№іеқҮ O(log N)пјҲжҹҘжүҫгҖҒжҸ’е…ҘгҖҒеҲ йҷӨпјүпјҢеӣ еұӮж•°жҳҜеҜ№ж•°зә§еҲ«гҖӮ
      • жңҖеқҸ O(N)пјҲжһҒз«ҜйҡҸжңәжғ…еҶөпјҢдҪҶжҰӮзҺҮжһҒдҪҺпјүгҖӮ
    • з©әй—ҙеӨҚжқӮеәҰпјҡ
      • O(N)пјҢжҜҸиҠӮзӮ№еҸҜиғҪжңүеӨҡеұӮжҢҮй’ҲпјҢдҪҶе®һйҷ…еҚ з”ЁжҜ”зҗҶи®әе°‘гҖӮ
